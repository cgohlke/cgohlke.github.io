DNA Curvature Analysis

by Christoph Gohlke

The online version of DNA Curvature Analysis is currently unavailable.

To use the same app on a personal computer, install Python and the dnacurve package:

python -m pip install -U dnacurve

Start the app on the command line

python -m dnacurve --web

or via the dnacurve_web executable in Python's script folder.
